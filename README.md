# 🧠 Mini CRM with OpenClaw AI Automation

This repository contains a **Mini CRM system** powered by **OpenClaw AI Agent**, along with setup guides for integrating automation and deploying the system.

The goal of this project is to demonstrate how AI agents can be used to **automate customer interactions, manage leads, and streamline workflows** using modern tools.
