# 🚀 OpenClaw Setup with Ollama

This guide walks you through setting up OpenClaw using Ollama and connecting it to Telegram for messaging.

---

## 📦 Prerequisites

- Ollama account  
- Telegram account  
- Ollama installed on your local machine  

---

## ⚙️ Setup OpenClaw with Ollama

1. Go to the Ollama website and create an account.  
2. Generate an API key and save it for later use.  
3. Install Ollama on your laptop.  
4. Run Ollama on your system.  
5. Select the option to launch **OpenClaw**.  
6. Set a name for your AI agent.  
7. Configure your agent based on your preferences.  

---

## 🔁 Running OpenClaw Again

For future use:

- Open Ollama  
- Select **Launch OpenClaw**  

---

## 📱 Connecting OpenClaw to Telegram

OpenClaw supports multiple communication channels such as:

- Telegram  
- WhatsApp  
- Gmail  

In this guide, we’ll use **Telegram**.

---

## 🤖 Create a Telegram Bot

1. Open Telegram and search for `@BotFather`.  
2. Start a chat and type: "/start"
3. Create a new bot: "/newbot"
4. Follow the prompts:
- Enter bot name  
- Enter bot username  
5. Copy the **bot token** provided by BotFather.  

---

## 🆔 Get Your Telegram User ID

1. Search for `@userinfobot` in Telegram.  
2. Start a chat.  
3. Copy your **user ID**.  

---

## 🔧 Configure OpenClaw for Telegram

1. Open OpenClaw settings.  
2. Enable the **Telegram channel**.  
3. Paste your **bot token**.  

---

## 🔐 Direct Message Policy

Choose one:

- **Pairing Mode** → Only you can message the bot  
- **Open Mode** → Anyone can message the bot  

---

## 👥 Group Message Policy

Choose one:

- **Allowlist** → Only selected users can interact  
- **Open Mode** → All group members can interact  

---

## ✅ Final Step

- Add your **Telegram User ID** to the allowed list for direct messages.  

---

## 🎉 You're Done!

Your OpenClaw AI agent is now connected to Telegram and ready to use.
